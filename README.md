## 使用指南

1. **GitHub 源码**: 表格中的“点击查看”链接对应本仓库中各章节的实战代码文件夹。您可以直接克隆本仓库到本地学习。
2. **Kaggle 在线运行**: 部分需要消耗 GPU 算力的章节，提供了配置好环境的 Kaggle Notebook 链接。您可以直接点击 `Copy & Edit` 在云端运行。
